Welcome to the LangGraph Platform reference docs. The following pages detail the interfaces you will use when building with LangGraph Platform:

* **[Environment Variables Reference](/langgraph-platform/env-var)**: Complete list of environment variables for configuring LangGraph Platform deployments.

* **[Python SDK Reference](/langgraph-platform/python-sdk)**: Python SDK for interacting with instances of the LangGraph Server.

* **[JavaScript/TypeScript SDK Reference](/langgraph-platform/js-ts-sdk)**: JavaScript/TypeScript SDK for interacting with instances of the LangGraph Server.

* **[Remote Graph Reference](/langgraph-platform/remote-graph)**: Documentation for using remote graphs and connecting to external LangGraph deployments.

* **[FAQ](/langgraph-platform/faq)**: Frequently asked questions and common troubleshooting information for LangGraph Platform.
